  - 与类组件不同，**函数组件通常更简洁**，并且在某些情况下性能更好。

  - 如果在调用某个组件时，需要传递一个布尔属性，并且值为 true，可以简写属性名，不需要写属性值。
    - 写法1：`<Comp boolProp />`
    - 写法2：`<Comp boolProp={true} />`
    - 上述两种写法等效。
  - 注意：如果直接在页面上渲染一个 true，需要将其转为字符串类型后才能渲染出来。
